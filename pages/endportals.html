END PORTAL TRIANGULATION









Fighting the Ender Dragon is hard, but finding the End in the first place can be just as difficult. This calculator, however, can estimate where a stronghold is based on two throws of an Eye of Ender using trigonometry and linear equations.

Do know that it can only give you an estimate, so repeating it at the output location is a good idea for more accuracy. Simply press the [How to Use?] button to find out how to use the calculator.
